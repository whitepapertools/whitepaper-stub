# flag

Блок для отображения флагов различных стран.

### Модификаторы блока

| Модификатор | Значение | Описание |
|------------ | -------- | -------- |
| `name` | `blr` `chn` `eur` `gbr` `kaz` `rus` `usa` | Имя флага |
| `size` | `s` `m` | Размер |
| `form` | `round` | Форма |

```js
{
	block: 'flag',
	mods: { name: 'rus', size: 's' }
}
```
