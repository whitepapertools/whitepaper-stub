# Блок payment-icon

Блок иконок платёжных систем

### Модификаторы блока

| Название | Значения | Описание |
| -------- | -------- | -------- |
| `name` | `visa_brand`, ...  | **Обязательный**. Имя платёжной системы |
| `size` | `s` `m` `l` | Размер |

```json5
{
	block: 'payment-icon',
	mods: { 'name': 'visa_brand', size: 's' }
}
```
