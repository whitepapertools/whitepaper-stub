# brand-logo

Блок для отображения логотипов.

### Модификаторы блока

| Название | Значения | Описание |
| -------- | --------- | ------- |
| `name` | `absolute` `acado` `accion`, ...| Имя иконки |
| `size` | `s` `m` `l` `xl` | Размер |

```js
{
	block: 'brand-logo',
	mods: { name: 'allo-inkognito', size: 's' }
}
```
